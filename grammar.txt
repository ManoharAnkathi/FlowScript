FlowScript Grammar (Front-End Scope)

program      -> START NL* stmt_list END NL* EOF
stmt_list    -> (statement NL+)*
statement    -> declaration | assignment | print_stmt
declaration  -> LET IDENT ASSIGN expression
			 | CONST IDENT ASSIGN expression
assignment   -> IDENT ASSIGN expression
print_stmt   -> PRINT ARROW expression

expression   -> term ((PLUS | MINUS) term)*
term         -> factor ((STAR | SLASH) factor)*
factor       -> NUMBER | IDENT | LPAREN expression RPAREN

Lexical Notes:
- START token maps to '@start'
- END token maps to '@end'
- ASSIGN token maps to ':='
- ARROW token maps to '=>'
- Comments start with '#'

Unique Front-End Features:
- const declarations are immutable and cannot be reassigned.
- typo-aware parser diagnostics suggest closest keywords/directives.
- semantic analysis warns for declared-but-unused variables.
- CLI supports profiling and JSON compile reports.
